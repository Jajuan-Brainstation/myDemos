flex-direction

Control the direction elements should flow and pick the axis you wish to use as the main axis

  • row (default): items run left to right along the x-axis
  • column: items run top to bottom along the y-axis
  • row-reverse: items run right to left along the x-axis
  • column-reverse: items run bottom to top along the y-axis

justify-content

Control spacing of elements along the main axis

  • flex-start (default): items are packed toward the start line
  • flex-end: items are packed toward the end line
  • center: items are centered along the line
  • space-between: first item at start, last item at end, even spacing between the rest
  • space-around: items are evenly distributed with one unit of space around each
  • space-evenly: items are distributed so that there is equal spacing on either side of each element

align-items

Control alignment of elements along the cross axis

  • flex-start (default): items are packed toward the start line
  • flex-end: items are packed toward the end line
  • center: items are centered along the line
  • stretch: items are stretched such that their cross sizes are all the same -- respecting width and height constraints
  • baseline: items are aligned such that their flex container baselines align

flex-grow, flex-basis and flex-shrink

Control the relative size of elements in flexbox container

  • flex-grow: flex grow factor, a unitless value defining a ratio of item's share of available space. Larger value means prioritize item's growth.
  • flex-shrink: flex shrink factor, a unitless value defining item's ratio of shrink priority. Larger value means prioritize item's shrinkage
  • flex-basis: sets the initial main size of a flex item. Following order with other size properties: content ‚Äî> width ‚Äî> flex-basis (limited by max | min-width)
  • flex: a shorthand property that allows to set flex-grow flex-shrink and flex-basis in a single value
